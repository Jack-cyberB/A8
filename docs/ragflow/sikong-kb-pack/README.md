## 暂不建议直接放进主知识库的内容

- 赛题 PDF、指导手册、直播答疑：更适合单独做“答辩与赛题说明”知识库
- `docs/design-reference/images/`：这是视觉参考，不适合知识检索
- BDG2 原始时序 CSV：这些数据应继续留给分析接口和统计逻辑处理，不建议交给 RAGFlow 做主检索
